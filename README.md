# React Router Prismatic Blog

## :bulb: Features
- **headless CMS** blog (content provided by prismic.io)
- posts are divided into categories
- use of **React Router** for accessing subpages
- pages had to be **paginated** with a limit of 5 posts by page 
- blog is responsive

## :gear: Installation

The project uses [node](https://nodejs.org/en/) and [npm](https://www.npmjs.com/).

- Clone the project using:

```bash
  git clone
```

- Go to the project directory and install all dependencies:

```bash
  npm i
```

- To run webpack and create a localhost in developer mode type:

```bash
  npm start
```
